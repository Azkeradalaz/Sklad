Инструкции по установке приложения Microsoft Visual Studio LightSwitch

 Ниже приводятся действия по установке двухуровнего клиентского приложения для настольных ПК:

1.    Установите на целевой (клиентский) компьютер экспресс-выпуск SQL Server, если он еще не установлен. Для приложений LightSwitch требуется SQL Server 2005 или более поздней версии либо экспресс-выпуск SQL Server 2005 или более поздней версии. Дополнительные сведения о параметрах установки экспресс-выпуска SQL Server.

2.    На целевом компьютере разверните схему базы данных с помощью средства командной строки sqlcmd.exe. Средство Sqlcmd.exe устанавливается с SQL Server и обычно находится в папке \Microsoft SQL Server\100\Tools\Binn, расположенной в папке Program Files или Program File (x86).

a.     В командной строке с повышенными привилегиями перейдите в каталог, содержащий вывод публикации, и введите команду следующего вида:

 sqlcmd.exe -i MyApplication.sql -S .\SQLExpress

 Вместо MyApplication подставьте имя приложения, а вместо .\SQLExpress - расположение базы данных на сервере.

3.    Найдите файл web.config в папке Application Files вывода публикации.

a.     Откройте файл web.config в "Блокноте" и измените подключение в разделе <connectionStrings>, указав правильные имя сервера и базы данных. Например, для записи следующего вида:

<add name="_IntrinsicData" connectionString="Data Source=SERVERNAME\SQLEXPRESS;Initial Catalog=MyApplication;Integrated Security=True;Pooling=True;Connect Timeout=30;User Instance=False" />

Замените SERVERNAME на имя сервера баз данных, а MyApplication замените на имя своего приложения.

4.    Запустите программу Setup.exe на целевом компьютере, чтобы установить приложение.
